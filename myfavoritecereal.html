Brain****(or as I prefer to call it, Branflakes) is an esoteric programming language created in 1993 by Urban Müller.

And I love it

So here's a bunch of Branflakes scripts
To run them, put them into any Branflakes compiler or interpreter
Or alternatively, if you despise using anything that you didn't make yourself, you could write one in literally any language(even Branflakes itself...)
